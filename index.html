Regra da Cascata

O CSS funciona como uma “cascata”: quando há mais de uma regra para o mesmo elemento, o navegador decide qual usar considerando três fatores:

  • Origem da regra (CSS do autor, do navegador ou do usuário).
  • Especificidade do seletor (quanto mais específico, maior prioridade).
  • Ordem de aparição (em caso de empate, vale a última regra escrita).

Especificidade

Tag (ex.: p {}) → peso baixo

Classe/pseudo-classes (ex.: .box {}, :hover) → peso médio

ID (ex.: #menu {}) → peso alto

!important → maior peso de todos

!important sobrescreve quase tudo (mas é recomendável evitar).

Combinação de Seletores

Espaço (.primeiro h1) → Seleciona todo h1 dentro de .primeiro (descendentes em qualquer nível).

Filho direto (.primeira > h1) → Seleciona apenas h1 que são filhos diretos de .primeira.

Irmão adjacente (.primeira h1 + h1) → Seleciona o h1 que vem logo após outro h1 dentro de .primeira.

Pseudo-classes

  • :first-child → Seleciona o primeiro filho de um elemento.
  • :last-child → Seleciona o último filho.
  • :nth-child(n) → Seleciona o filho na posição n (ex.: nth-child(2) = segundo filho).
  • :nth-of-type(n) → Igual ao anterior, mas conta apenas filhos do mesmo tipo de tag.

Display Inline x Block

Block:

  • Ocupa toda a largura disponível.
  • Quebra linha automaticamente.
  • Permite aplicar width, height, margin, padding em todas as direções.

    • Ex.: div, p, h1.

Inline:

  • Ocupa apenas o espaço do conteúdo.
  • Não quebra linha automaticamente.
  • Só aceita margin horizontal (left/right) e padding horizontal, mas não altura (top/bottom).

    • Ex.: span, a, strong.

Elementos Padrão

Block: div, p, h1-h6, ul, ol, li, form.

Inline: span, a, img, strong, em, label.

Margin, Padding e Border

Margin → Espaço externo (fora da borda).

Padding → Espaço interno (entre o conteúdo e a borda).

Border → Linha que envolve o elemento.

Shorthand:

1 valor → aplica em todos os lados.

Ex.: margin: 10px;

2 valores → 1º = cima/baixo, 2º = esquerda/direita.

Ex.: padding: 10px 20px;

3 valores → 1º = cima, 2º = lados, 3º = baixo.

Ex.: margin: 10px 20px 30px;

4 valores → cima, direita, baixo, esquerda (sentido horário).

Ex.: border-width: 10px 5px 15px 0;

Sufixos:

  • -top, -bottom, -left, -right → controlam lados específicos.
  • -inline → controla margens/paddings horizontais.
  • -block → controla margens/paddings verticais.
